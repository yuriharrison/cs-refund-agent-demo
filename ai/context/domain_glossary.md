# Domain Glossary

- **Agentic Flow:** The sequence of autonomous decisions and function calls made by the AI.
- **System Message:** Visual confirmation of an action taken, distinct from a regular chat bubble.
- **Debug Message:** A special type of system message visible only in the demo to illustrate AI thought process and actions.
- **Escalation:** The process of the AI handing over the conversation to a human support agent when it cannot resolve the issue.
- **Snapshot Pattern:** A testing approach where real API responses are saved to disk on the first run and replayed in subsequent runs to avoid live API costs and latency.
